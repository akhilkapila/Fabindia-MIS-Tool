### Blue (Sales)

- **Impact**: Positive, achievable

## Color Codes for Reference

When customizing or extending:
```css
/* Primary Colors */
--color-sales: #007bff;      /* Blue */
--color-advances: #fd7e14;   /* Orange */
--color-banking: #6f42c1;    /* Purple */
--color-combine: #17a2b8;    /* Teal */
--color-final: #28a745;      /* Green */

/* Background Colors */
--bg-sales: #f0f7ff;         /* Light Blue */
--bg-advances: #fff8f0;      /* Light Orange */
--bg-banking: #f8f5ff;       /* Light Purple */
--bg-combine: #f0f8ff;       /* Light Blue */
--bg-final: #f0fff4;         /* Light Green */
```

## Why This Color System Works

1. **Distinct**: Each color is visually different
2. **Memorable**: Users remember which tab is which color
3. **Professional**: Maintains clean, modern appearance
4. **Accessible**: Sufficient contrast ratios
5. **Scalable**: Can be extended for future features
6. **Consistent**: Same color used throughout section
7. **Intuitive**: Color naturally flows through process
